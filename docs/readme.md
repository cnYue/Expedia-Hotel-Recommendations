## 关于kaggle比赛
一般来说 kaggle的基本套路就是 

处理数据 -> 使用各种工具和模型训练数据 -> 上传结果 根据结果的好坏进行调参 -> 将各种模型下的结果进行合并 得出一个ensemble model(就翻译为联合模型好了) -> 调整最终模型的参数 -> 谋事在人 成事在天
